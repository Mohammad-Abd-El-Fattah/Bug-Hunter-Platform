    Tips & Tricks

    Reconnaissance
    Subdomain Enumeration

    Use multiple tools and sources for comprehensive results:

    subfinder -d target.com -all | httpx -silent
    Certificate Transparency

    Search certificate transparency logs:

    curl -s "https://crt.sh/?q=%.target.com&output=json"
    DNS Bruteforcing

    Use custom wordlists for better results:

    massdns -r resolvers.txt -w results.txt wordlist.txt
    Vulnerability Testing
    XSS Testing

    Test for XSS in all input fields and parameters:

    <script>alert('XSS')</script>
    SQL Injection

    Start with basic payloads:

    ' OR '1'='1
    SSRF Testing

    Test for Server-Side Request Forgery:

    http://localhost:80/admin
    Best Practices
    Documentation

    Always document your findings with:

    • Steps to reproduce
    • Impact assessment
    • Screenshots/videos
    Scope Verification

    Always verify targets are in scope before testing

    Responsible Disclosure

    Follow responsible disclosure practices and program guidelines

    Advanced Techniques
    Automation Scripts

    Create bash scripts for repetitive tasks:

    #!/bin/bash
for domain in $(cat domains.txt); do
    nmap -sV -T4 $domain
    nuclei -target $domain
done
    Custom Payloads

    Maintain a personal payload collection:
